## 🚀 Projetos em Desenvolvimento
### 🏋️ Sistema de Aulas Funcionais
<p align="center">
	<a href="">

</a>
	<a href="https://github.com/RafaelOlive26sp">
      <img src="https://img.shields.io/badge/status-em%20desenvolvimento-yellow"/>
    </a>
	<a href="www.linkedin.com/in/rafael-delgado-728211197">
      <img src="https://img.shields.io/badge/vers%C3%A3o-0.1.0-blue"/>
    </a>
</p>

Sistema web completo para gerenciamento de aulas funcionais, permitindo agendamentos, controle de frequência, avaliação de desempenho e relatórios personalizados.

### ✨ Funcionalidades principais

🔐 Autenticação multi-nível (alunos, professores, administradores)<br>
📅 Agendamento de aulas com notificações<br>
📊 Dashboard personalizado com Tutorial para ajudar o Professor<br>
📝 Relatórios detalhados para professores e gestores

### 🛠️ Stack Tecnológica
Front-end:

Vue 3 com Composition API
Vuetify 3.0 para interface responsiva
Pinia para gerenciamento de estado
Vite como build tool


Back-end:

Laravel 11
Sanctum para autenticação via API tokens
MySQL 8.0
Eloquent ORM para manipulação de dados
Aplicado SOLID para boas práticas de desenvolvimento
